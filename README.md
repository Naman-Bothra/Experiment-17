# Theory : 

 Matplotlib is a popular Python library used for data visualization—it helps you create graphs, charts, and plots to understand data easily.

 Seaborn is a Python library built on top of Matplotlib that is used for advanced and attractive data visualization with less code.

## What Matplotlib Does

It allows you to turn raw data into visuals like:

Line graphs (trends over time)

Bar charts (comparisons)

Pie charts (proportions)

Histograms (data distribution)

Scatter plots (relationships between variables)

## Why It’s Used

Makes data easier to understand

Helps identify patterns and trends

Widely used in data science, EDA, and research

Import matplotlib.pyplot as plt: Imports the primary library used for creating static, interactive, and animated visualizations in Python.

import seaborn as sns: Imports a high-level interface based on matplotlib for drawing attractive and informative statistical graphics.

# What Seaborn Does

Seaborn helps you create more beautiful and informative graphs such as:

Heatmaps (correlation matrices)

Box plots (data distribution)

Violin plots

Pair plots (relationships between multiple variables)

Enhanced bar and line charts

# Why Use Seaborn

Better-looking graphs by default

Easier to use than Matplotlib for statistical plots

Works directly with Pandas DataFrames

Built-in themes and color palettes

# Commands used in the Experiment:

1.plt.bar(): Creates a bar chart to compare different categories of data using rectangular bars.

2.plt.xlabel() / plt.ylabel(): Sets the text labels for the x-axis and y-axis respectively to provide context to the plot.

3.plt.title(): Adds a main heading to the top of the chart to describe the visualization.

4.plt.show(): Renders and displays all currently active figure objects onto the screen.

5.plt.hist(): Generates a histogram to represent the distribution of a continuous numerical variable by binning data.

6.plt.scatter(): Produces a scatter plot to observe the relationship or correlation between two numerical variables.

7.plt.pie(): Creates a pie chart to illustrate numerical proportions as slices of a circle.

8.plt.legend(): Places a box on the graph to describe the different elements or data series plotted.

9.sns.lineplot(): Draws a line graph to show data trends over a continuous interval or time period.

10.sns.countplot(): Uses bars to show the frequency of observations in each categorical bin.

11.sns.boxplot(): Displays the distribution of data based on a five-number summary (minimum, first quartile, median, third quartile, and maximum).
